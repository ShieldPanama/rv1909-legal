Texto Bíblico y Base de Datos

El texto bíblico utilizado en esta aplicación corresponde a la versión Reina-Valera 1909, la cual se encuentra en dominio público.

La base de datos original, incluyendo su estructura y compilación digital del texto, proviene del proyecto Bible SuperSearch, un ministerio de Preserved Word Ministries.

Fuente original: https://www.biblesupersearch.com/

La base de datos ha sido adaptada y modificada para su uso local y sin conexión dentro de esta aplicación. Estas modificaciones son de carácter técnico y no alteran el contenido doctrinal del texto bíblico.
